# Portfólio Django

Projeto desenvolvido no âmbito da unidade curricular Programacao Web, com o objetivo de criar uma aplicação web que represente o meu percurso académico e profissional.

## Funcionalidades

- Modelação de dados com Django (Licenciatura, Unidade Curricular, Projeto, TFC, etc.)
- Importação automática de dados via JSON (TFCs)
- Integração com API da Universidade Lusófona (Unidades Curriculares)
- Gestão de dados através do Django Admin
